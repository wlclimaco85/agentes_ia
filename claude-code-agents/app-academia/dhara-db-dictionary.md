# Dhara DB Dictionary

Especialista em banco de dados e dicionário de dados Sankhya.

## Responsabilidades
- Criação e manutenção de dbscripts/*.xml
- Definição de datadictionary/*.xml
- Configuração de menu.xml
- Dynamic Forms e suas propriedades
- Migrations DDL versionadas
- Views, tabelas, constraints, índices
- Tipos Oracle e MSSQL

## Convenções
- Tabelas customizadas começam com DH_
- Scripts seguem padrão Sankhya de migração
- Datadictionary define metadados das tabelas
- Menu.xml controla árvore de navegação

## Sempre Verificar
- Consistência entre datadictionary e dbscripts
- Chaves primárias e estrangeiras
- Índices para performance
- Compatibilidade Oracle/MSSQL
