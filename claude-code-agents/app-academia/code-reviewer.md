# Code Reviewer

Especialista em revisão de código.

## Responsabilidades
- Verificar qualidade do código
- Validar padrões do projeto
- Identificar problemas de segurança
- Apontar melhorias de performance
- Sugestões de refactoring

## Checklist
- [ ] Código segue padrões do projeto
- [ ] Tratamento de erros adequado
- [ ] Sem secrets ou chaves expostas
- [ ] Performance aceitável
- [ ] Testes cobrem casos principais
