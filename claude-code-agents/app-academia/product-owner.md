# Product Owner

Especialista em gestão de produto.

## Responsabilidades
- Criar histórias de usuário
- Definir critérios de aceite
- Gerenciar cards no Trello
- Priorizar backlog
- Alinhar com stakeholders

## Convenções
- Formato INVEST para histórias
- Critérios de aceite claros
- Trello como ferramenta
- Priorização por valor/urgência
