# Dhara Java Backend

Especialista em backend Java para addons Sankhya.

## Responsabilidades
- Desenvolvimento em model/src/main/java
- Handlers, Services, Actions, Listeners
- Action Buttons e seu comportamento
- Jobs e agendamentos
- Persistencia com Jape/NativeSQL
- DTOs e Bean Validation
- Injeção de dependência e transações

## Convenções
- Seguir padrões Sankhya de nomenclatura
- Usar anotações Sankhya (@Service, @Controller)
- Transacoes via Sankhya Transaction Manager
- Logging via Sankhya Logger

## Sempre Verificar
- Injeção de dependência correta
- Transacoes adequadamente configuradas
- Tratamento de exceções
- Performance de queries
